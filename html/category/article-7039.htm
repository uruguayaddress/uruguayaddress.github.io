吸血鬼的电影叫什么（吸血鬼第五季剧情）不要告诉别人，

今天君君给大家分享一波吸血鬼电影。尺度与颜值齐飞，绝对上头！值得收藏，赶快码住！

1.《夜访吸血鬼》1994豆瓣评分：8.3主演：布拉德·皮特 / 汤姆·克鲁斯推荐指数：★★★★★

改编自安妮·赖斯的《吸血鬼编年史》系列第一部。庄园主路易在妻女死去后痛不欲生，遇到了吸血鬼莱斯特，被其施法变成了同类。从此，两人开始了相爱相杀的生活。

阿汤哥和布拉德皮特，史上颜值最高的吸血鬼组合！

2.《惊情四百年》1992豆瓣评分：7.7主演：加里·奥德曼 / 薇诺娜·瑞德推荐指数：★★★★

吸血鬼传说+凄美爱情故事。改编自布拉姆的吸血鬼小说《德古拉》。

公元1462年，土耳其人入侵君士坦丁堡，并威胁到整个基督教大将军德古拉伯爵临危受命征讨，征战过程中谣言四起他的妻子以为他已经战死，悲痛欲绝自杀，由此他迁怒上帝，投入邪恶势力，以吸食鲜血延续生命，成了一个不死的吸血僵尸。

3.《吸血鬼生活》2014豆瓣评分：7.9主演：杰梅奈·克莱门特 / 塔伊加·维迪提推荐指数：★★★★

一群逗比吸血鬼的日常。除了要定时吸点血外，他们就是一群神经病。他们生气时会飞起来打架，进食时不小心搞得满地是血，还得为大扫除而喋喋不休……

此外还要应付人类社会的繁杂规矩，例如付房租、设法混进夜店、处理室友矛盾等等。惊悚与搞笑并行，极具特点。

4.《唯爱永生》2013豆瓣评分：7.6主演：蒂尔达·斯文顿 / 汤姆·希德勒斯顿推荐指数：★★★★

这是一对吸血鬼恋人的故事。男女主是携手共同穿越了数百年时光的吸血鬼恋人，从中古走到现代，两人经历了人类社会的数度大起大落。

电影没有太多血腥恢弘的场面，吸血鬼一直沉浸在美好的小世界中“抖森”的美颜和忧郁气质无疑是一大看点5.《暮光之城》2008豆瓣评分：7.6主演：克里斯汀·斯图尔特 / 罗伯特·帕丁森推荐指数：★★★★

吸血鬼和人类的爱情故事。因为父母离异，贝拉随父亲来到小镇生活。在新学校里，贝拉留意到一群举止诡异的怪人，他们特立独行，很是神秘。

在实验课上，她遇到了怪人中的一个，金发帅哥爱德华。爱德华和她交往并坦白，自己是素食吸血鬼，一场人鬼之恋由此展开……

6.《生人勿进》2008豆瓣评分：7.6主演：凯尔·赫德布朗特 / 莉娜·林德尔森推荐指数：★★★★

吸血鬼小萝莉一个遭遇校园霸凌的小男孩喜欢yy各种虐杀犯罪案件，而隔壁邻居家新搬来的小孩爱莉，她走进了奥斯卡的生活，两人慢慢成为朋友，但他却发现小女孩身上有着不可告人的恐怖秘密。

虽然有点儿小恐怖，但又不失温情。

7.《日班猎人》2022豆瓣评分：6.5主演：杰米·福克斯 / 戴夫·弗兰科推荐指数：★★★

血浆量和动作量十足，吸血鬼版《疾速追杀》。男主表面上是一个泳池清洁工，真实身份是吸血鬼猎人，吸血鬼的獠牙非常值钱，他靠此维持生计。

在猎杀吸血鬼的过程中，他得罪了吸血鬼女王，一场巨大的危机即将来临，他能否化险为夷？

8.《范海辛》2004豆瓣评分：7.2主演：休·杰克曼 / 凯特·贝金赛尔推荐指数：★★★★

吸血鬼与狼人。这部电影涉及的东西很多。里面有吸血鬼、狼人、吸血鬼猎人、科学怪人等等。人物的设定还是来源于著名小说《德古拉》。

故事讲述的是，在十九世纪的罗马尼亚。出现了吸血鬼德古拉和他领导的邪恶团队。为了铲除这一恶势力，便请来了吸血鬼猎人范海辛。范海辛为了制服德古拉还变成了狼人。

9.《黑夜传说》2003豆瓣评分：7.5主演：凯特·贝金赛尔 / 斯科特·斯比德曼推荐指数：★★★★

哥特风格的吸血鬼狼人动作片。吸血鬼和狼人是两大互相争斗的种族。

女主是吸血鬼族的月神。她为了阻止狼人族的阴谋救了男主。结果发现男主有着吸血鬼和狼人的双血统。主要剧情是吸血鬼和狼人的争斗打杀。

10.《蝙蝠》2009豆瓣评分：7.4主演：宋康昊 / 金玉彬 / 金海淑推荐指数：★★★★

韩国吸血鬼电影，朴赞郁执导。原本受人尊敬的神父因意外感染病毒而变成奇怪的吸血鬼，与年龄相当于自己女儿的年轻少妇发生不伦恋情。

直到阳光将你我化为灰烬，爱情是拯救还是毁灭？

11.《刀锋战士》1998豆瓣评分：7.1主演：韦斯利·斯奈普斯 / 斯蒂芬·多尔夫推荐指数：★★★★

半吸血鬼与吸血鬼王的斗争剧情比较简单，但炫酷耍宝的打斗场面以及毫不避讳的血浆横飞，还是让这部电影保持了不错的评价。

看完电影，君君有一个问题：如果人类都变成吸血鬼，那么他们吸什么？

吸血鬼三姐妹来到柏林，大姐露易丝发现了靠偷窃为生的假小子丽娜，把她变成了其中一员，开始了奢靡的生活。

全员吸血鬼病毒席卷全球，吸血鬼统治了世界，人类沦为圈养的…仅存的人类被圈养起来为它们制造新鲜血液，但当“食物”数量越来越少，新的危机也随之来临。

恐怖分子劫持飞机，却遭遇吸血鬼。机组200人中，谁能笑到最后？

设定新奇带感，氛围紧张刺激，还有不少动作戏。可惜口碑两极分化，纯的朋友可入坑！
